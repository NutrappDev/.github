# NutrappDev

Organización de desarrollo de software de **Nutrabiotics** / **NutrApp**

Aquí vive el código que impulsa nuestros productos digitales: APIs, integraciones, frontend y automatizaciones que conectan a nuestros equipos con nuestros usuarios.

---

## ¿Qué construimos?

- Plataformas de gestión nutricional y seguimiento de salud
- APIs e integraciones con sistemas de terceros
- Automatizaciones de procesos internos
- Herramientas internas para equipos médicos y comerciales

---

## Estándares de desarrollo

Todos nuestros repos siguen estándares centralizados definidos en este repositorio:

- **Flujo de ramas:** `feat/fix/...` → `develop` → `qa` → `main`
- **Commits:** [Conventional Commits](https://www.conventionalcommits.org/) con ticket Jira obligatorio
- **PRs:** validados automáticamente por GitHub Actions
- **Releases:** trazables desde Jira hasta producción
